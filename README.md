**CYBER GUARD**
I made a custom password generator where:
-User can decide if they have to include uppercases, lowercases, digits, special characters in the password or not.
-User can also decide the length of the password (min-5 & max-44)
-User can copy the password.
-User can access to the lsat 5 previous generated passwords.
-User can access to the last 5 copied passwords.
-User can know the strength of the password.

*TECH STACK*
-HTML : structure
-CSS : designing
-JS : main logic

*How to use*
-Go to "https://reaperly.com/Password/"
-Check the boxes you want.
-Select the password length.
-Click generate.
-Can copy the password by copy button.
-Change the password if doesn't like the password.
